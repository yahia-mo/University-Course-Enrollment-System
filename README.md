# University Course Enrollment System

A CLI system to manage students, admins, courses, and subject enrollment in a university.

---

## Features

- Student registration and login
- Admin management
- Course enrollment system
- Credit hour limit control
- Secure password hashing
- SQLite database

---

## Technologies Used

- Python 3
- SQLAlchemy
- SQLite

---

## How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/yahia-mo/University-Course-Enrollment-System.git
cd University-Course-Enrollment-System
```

### 2. Install The Requirments

```bash
pip install -r requirements.txt 
```

## How To use

- The app is Ready to use and Add your own Data .
- In First Use you must use the Defult Admin Account .
